# Schlaf-Tracker 🌙

Eine einfache App, um zu tracken, wann unser Sohn einschläft und aufwacht.

## Funktionen

- **😴 Eingeschlafen** – speichert mit einem Tipp die aktuelle Uhrzeit
- **☀️ Aufgewacht** – speichert mit einem Tipp die aktuelle Uhrzeit
- **✏️ Zeit manuell eintragen** – Uhrzeit und Datum frei wählen (falls man das Tippen mal vergisst)
- Verlauf nach Tagen gruppiert (Heute / Gestern / Datum)
- Einträge lassen sich löschen
- Alle Daten werden lokal im Browser gespeichert (localStorage) – kein Server, kein Login

## Nutzung

`index.html` einfach im Browser öffnen. Läuft komplett offline auf dem Handy oder am Rechner.

Für den Handy-Homescreen: Seite öffnen und über das Browser-Menü „Zum Startbildschirm hinzufügen".
